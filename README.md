## Installation

Add to your project using the package manager of your preference:

```bash
npm install --save rescript-polished
# or yarn:
yarn add rescript-polished
```

Add `rescript-polished` as a `bs-dependency` in your bsconfig.json:

```json
"bs-dependencies": [
  "rescript-polished"
]
```
